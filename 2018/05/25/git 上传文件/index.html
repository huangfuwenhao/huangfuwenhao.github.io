第一步 安装 git

  • 打开命令行窗口 win + R 写 cmd 回车，打开是为了在一开始的时候配置一个全局的邮箱和用户名。
  • 输入 git config –global user.email “邮箱” 写完敲击回车。
  • 输入 git config –global user.name “用户名” 写完敲击回车。
  • 为了看一下配置文件列表 git config –list 敲击回车，最后两个是刚才配置的用户名和邮箱。
  • 以后的操作就不用管了。

第二步 申请一个 github 账号

  • 新建一个仓库 new Repository。
  • Repository name 是你当前仓库的名称。
  • Description (optional) 当前仓库的说明（可以不写）。
  • 点击 create 创建成功。

第三步 初始化文件夹

  • 在本地新建一个文件夹，进到文进夹里面，右键单击鼠标 点击 git bash here
    会打开一个类似于命令行一样的窗口。
  • 输入 git init 敲击回车 – 把当前文件夹当作一个本地仓库（初始化成为一个仓库）。
  • git add . – 向暂存区提交文件。
  • git commit -m “shuoming” – 是对本次提交的说明。
  • git remote add origin https://github.com/WDBBdpd223322/heima44.git – 连接github的远程仓库。
  • git push origin master – 把暂存区里面的文件推送到远程仓库。

第四步 文件上传操作

  • 以后修改了本地仓库中的文件，哪怕是修改了已经提交的文件也要从新推送。
  • 在当前仓库中 右键点击 git bash here 。
  • git add . 。
  • git commit -m “本次提交的说明”。
  • git push origin master 。

注意

  • 一个本地仓库对应一个远程仓库。
  • 用户名密码输入的时候别写错。
  • 本地仓库的 .git 不要动。
  • 已经建立好的本地仓库不要改名。
